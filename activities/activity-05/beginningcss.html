Explain the difference between inline, embedded, and external CSS.

Inline CSS is applied directly inside an HTML element using the style attribute. It only affects that single element.

Embedded CSS (what I used for coloring the questions) is written inside a style tag in the head section of an HTML file, it only applies to that single page only.

External CSS is written in a separate .css file. It can be linked to multiple HTML pages.

Explain why you should only use external CSS.

You should only use external CSS is because it makes the code cleaner and easier to read. One file can also control the design of many pages which can be easier to update and maintain.

What is a selector used for?

A selector tells CSS which HTML elements to style.
