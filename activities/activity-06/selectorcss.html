1. List out the different ways to select an item

There are many different ways, some examples include Class and ID.

2. What is the difference between ID and a Class

The ID can only be used once on a page. Class selector can be put on the page as many times as you want.

3. What is the difference between a SPAN tag and a DIV tag?

The div is for block, and span is for inline

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

The div is general, and the other three have structured meaning, aka header, main, or footer section.

5. What are the different ways to select a color?

You can select the preset colors or create your own using RGBA scale.
